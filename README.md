# Brain_Tumour_Detection

It's a repo which contains the full project which is capable to detect the Brain Tumour from Brain MRI and it is developed using Python, Flask and Machine Learning.

### Installaion of required Libraries

For Installation of Libraries you must type the given command in your terminal.

pip install -r requirements.txt

### Run the project

Remember dataset is given in the data folder for training as-well-as for testing purpose. <br>
Simple run the app.py file.

python app.py
